# PSP Chat Multiusuario

Enlace al repositorio: [Reto_Chat_Multiusuario](https://github.com/Mosi-HB-2006/Reto_ChatMultiusuario.git)

## Pasos de ejecución

1. Clona el repositorio (git clone https://github.com/Mosi-HB-2006/Reto_ChatMultiusuario.git)
2. Abre el proyecto clonado en el paso anterior en NetBeans.
3. Cuando el proyecto esté abierto, ejecuta el servidor ubicado en el paquete server (clic derecho en la clase Servidor y seleccionar la opción Ejecutar).
4. Una vez abierto el servidor, se debe ejecutar el main para iniciar sesion con todos los usuarios deseados.
5. Para iniciar un usuario, introduce su nombre; luego la IP (en local sería "localhost") y el puerto por defecto es 5000 (esto se puede configurar en la clase Servidor si quieres).
6. Con los usuarios iniciados, se puede mandar un mensaje escribiendo el texto en el textfield de abajo a la izquierda, además de seleccionar si el mensaje es público o privado.
   6.1. Si el mensaje es privado, se debe seleccionar el destinatario en el combobox de abajo a la derecha de la ventana.
7. Una vez escrito el mensaje, pulsar el botón Enviar para publicarlo.
